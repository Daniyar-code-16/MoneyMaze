01 — What is MoneyMaze?

❓

The Problem

74% of teens don't feel confident managing money. Schools teach algebra but not how to file taxes or avoid credit card debt. Financial illiteracy costs people billions.

💡

The Solution

MoneyMaze turns financial education into an engaging adventure. Players navigate real-life scenarios: paying rent, investing in stocks, handling unexpected expenses — without real-world consequences.

✨

What Makes Us Unique

Adaptive difficulty, local economic contexts (Kazakhstan, US, Europe), and multiplayer challenges. Learn by doing, not by reading boring textbooks.

🎯 Target Audience

🧑 Teens 13-17

Learn budgeting basics before earning their first paycheck

👩‍🎓 Young Adults 18-25

Master credit scores, loans, and investments

👪 Families

Play together and discuss money management openly

02 — Functional Requirements

🎮 Game Mechanics

  • Monthly budget simulation with random events
  • Investment mini-game (stocks, crypto, real estate)
  • Credit score system that affects gameplay
  • Loan shark encounters (high interest rates!)
  • Tax season challenges
  • Achievements for financial milestones

📚 Educational Features

  • Interactive glossary of financial terms
  • Real-life scenarios (car purchase, medical bill)
  • Progress tracking for parents/teachers
  • Difficulty levels: Beginner to Expert
  • Localized content for different countries

👤 User System

  • User registration and profiles
  • Cloud save for progress
  • Multiplayer leaderboards
  • Parental controls and reports
  • Multi-language support (EN/RU/KZ)

03 — Tech Stack

Python 3.11 Pygame CE Flask (API) React (Dashboard) PostgreSQL SQLite Docker GitHub Actions Figma Aseprite

🎨 Frontend

Game Engine: Pygame CE for smooth 2D gameplay

Art Style: Modern flat design with financial aesthetics

Dashboard: React with Chart.js for progress visualization

⚙️ Backend

API: Flask RESTful with JWT auth

Database: PostgreSQL for production, SQLite for dev

Analytics: Custom tracking for educational progress

🚀 Deployment

Game: Itch.io + GitHub Releases

Website: GitHub Pages

API: Railway / Render
